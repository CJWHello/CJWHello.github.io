# LLM 卡片自动生成测试

这是一篇示例笔记，用来验证三件事：

1. `notes/llm/` 这种新类别是否会被识别。
2. Notes 页顶部是否会自动出现 `LLM` 分类按钮。
3. 点击卡片后是否能进入对应的 Markdown 详情页。

如果你在 GitHub 上 push 之后看到了新的 `LLM` 筛选按钮，并且能点进这篇笔记，说明整条自动生成链路是通的。
